# nlp100-python-roo

Roo Codeを使ってPythonで言語処理100本ノックを解きます。

## ディレクトリ構造

```
nlp100-python-roo/
├── .clinerules         # Roo Clineのルール設定ファイル
├── .gitignore          # Gitの除外ファイル設定
├── LICENSE             # ライセンスファイル
├── README.md           # このファイル
├── pyproject.toml      # プロジェクト設定ファイル
├── src/                # ソースコードディレクトリ
│   └── nlp100/         # メインパッケージ
│       ├── __init__.py # パッケージ初期化ファイル
│       ├── ch01/       # 第1章のソースコード
│       │   ├── __init__.py
│       │   ├── q00.py  # 問題00の解答
│       │   ├── q01.py  # 問題01の解答
│       │   └── ...     # その他の問題
│       ├── ch02/       # 第2章のソースコード
│       └── ...         # その他の章
└── tests/              # テストディレクトリ
    ├── __init__.py     # テストパッケージ初期化ファイル
    ├── ch01/           # 第1章のテスト
    │   ├── __init__.py
    │   ├── test_q00.py # 問題00のテスト
    │   ├── test_q01.py # 問題01のテスト
    │   └── ...         # その他の問題のテスト
    ├── ch02/           # 第2章のテスト
    └── ...             # その他の章のテスト
```

## テストの実行方法

```bash
# すべてのテストを実行
pytest

# 特定の章のテストを実行
pytest tests/ch01/

# 特定の問題のテストを実行
pytest tests/ch01/test_q00.py

# カバレッジレポートを生成
pytest --cov=nlp100
